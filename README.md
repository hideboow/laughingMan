# laughingMan
=====================================
概要　　
webカメラを使って、攻殻機動隊の「笑い男」になれるコードです。　　
OpenCVを使って顔認識を行い、笑い男の画像を認識箇所に当てはめています。  
笑い男は回転するリングパーツと、回転しないフェイスパーツに分けています。  

格納ファイル説明　　
=====================================
dataフォルダ:笑い男のパーツ画像　
laughingMan.pde:processing実行ファイル　


動作環境  
=====================================
mac OSX:10.10.1  
processing :2.2.1  

必要なライブラリ    
=====================================
OpenCV
